# Synthetic Data

Synthetic CSV files are generated by scripts in `src/generate` and orchestrated by `src/run_pipeline.py`.

## Default row volumes

- employees: 1,000
- projects: 5,000
- timesheets: 40,000
- employee_position_history: variable by employee history depth

## Included edge cases

- Position-history gaps, overlaps, and same-day collisions
- Duplicate project codes with newer updates
- Expected duplicate timesheet business rows
- Boundary-date timesheet entries for temporal logic testing

## Regeneration

Run:

`python src/run_pipeline.py --seed 42 --employees 1000 --projects 5000 --timesheets 40000`
